## Implementation

1. Prepare evaluation dataset: export color-annotated crops from CVAT "attribute-eval" project
2. Candidates: ResNet-18 vs EfficientNet-B0 color classifiers
3. For each candidate:
   - Export to ONNX → TensorRT FP16 (or ONNX for CPU pilot)
   - Deploy to Triton model repo
   - Run on evaluation dataset
   - Log to MLflow experiment 'attribute-bakeoff'
4. Metrics: per-color accuracy, confusion matrix, unknown_rate (% classified as unknown), latency_ms, throughput
5. Generate comparison report

- scripts/bakeoff/prepare_color_eval_data.py — export from CVAT, organize by color
- docs/bakeoff-results/attribute-comparison.md — results table + recommendation
