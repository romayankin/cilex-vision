# P3-A02: Re-ID Training Data Collection

You are working in /repo/scripts/annotation/. Curate cross-camera identity pairs for Re-ID model training.

## Context

To improve the Re-ID model, we need curated training data of confirmed cross-camera identity pairs.

## Reading Instructions

Before writing any code, read:
- .agents/handoff/P2-A01.md — cross-camera annotation setup
- .agents/handoff/P2-V02.md — MTMC service, confirmed associations
- .agents/handoff/P2-E02.md — MTMC evaluation results
- CONVENTIONS.md

## Implementation

1. Create scripts/annotation/collect_reid_training_data.py:
   - Export high-confidence MTMC associations (>0.9 confidence) as positive pairs
   - Mine hard negatives: same camera, different identity, similar appearance
   - Export as triplet format: (anchor, positive, negative) with crop URIs
2. Create scripts/annotation/validate_reid_pairs.py:
   - Sample pairs for human validation
   - Create CVAT task with pair images side-by-side
   - Track validation accuracy
3. Create scripts/annotation/reid_dataset_builder.py:
   - Build final Re-ID training dataset from validated pairs
   - Version with DVC
   - Split into train/val/test

## Constraints

- Only use high-confidence associations as positive examples
- Must validate a sample before using for training
- Must pass `ruff check`
