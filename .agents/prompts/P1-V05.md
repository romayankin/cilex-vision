# P1-V05: Metadata Bulk Collector

You are working in /repo/services/bulk-collector/. Python service using asyncpg.

- main.py — Kafka consumer group, batch messages, flush to TimescaleDB.
- collector.py — accumulate in memory buffer. Flush when batch_size (1000) or max_age_ms (500ms) reached.
- writer.py — asyncpg connection pool, COPY protocol (copy_records_to_table). Deduplicate by (camera_id, frame_seq, local_track_id).

CRITICAL: COPY protocol only, never row-by-row INSERT. Target: sustain 5,000+ rows/sec.

Metrics: rows_written_total, batch_size_histogram, write_latency_ms, write_errors_total, consumer_lag.
Tests: test_collector.py (batching logic), test_writer.py (COPY works with test DB).
Load test: scripts/load-test/load-test-collector.py — 100-camera equivalent (5000 det/sec) for 5 minutes. Assert no data loss.
