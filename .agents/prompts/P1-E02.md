# P1-E02: Tracker Configuration Bake-Off (5 days)

You are working in /repo/scripts/bakeoff/. Using the detector selected in P1-E01.

1. Prepare MOT-format ground truth from CVAT-annotated pilot clips.
2. Run each tracker config (ByteTrack, BoT-SORT) on same video sequences.
3. Compute via py-motmetrics: MOTA, IDF1, ID_switches, fragmentations, mostly_tracked, mostly_lost.
4. Log all runs to MLflow experiment 'tracker-bakeoff'.
5. Generate: docs/bakeoff-results/tracker-comparison.md.

Scripts:
- scripts/bakeoff/run_tracker_bakeoff.py --tracker bytetrack --detector {winner} --gt data/eval/mot/
- scripts/bakeoff/run_tracker_bakeoff.py --tracker botsort --detector {winner} --gt data/eval/mot/
- scripts/bakeoff/compare_bakeoff.py --experiment tracker-bakeoff --output docs/bakeoff-results/tracker-comparison.md
