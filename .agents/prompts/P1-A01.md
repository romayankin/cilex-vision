# P1-A01: CVAT Setup & Annotation Baseline

You are working in /repo/scripts/annotation/. Create:

1. setup_cvat_projects.py — CVAT API script to create:
   - Project "detection-eval" with labels from taxonomy.md
   - Project "tracking-eval" with track mode, MOT export
   - Project "attribute-eval" with color attribute labels

2. docs/annotation-guidelines.md — labeling guide with:
   - Bounding box rules, track mode rules, color attribute rules
   - Quality criteria (IoU >0.7 with ground truth)
   - Visual examples

3. compute_iaa.py — inter-annotator agreement:
   - Box IoU agreement, class agreement (Cohen's kappa), color agreement (Fleiss' kappa)
   - Output: IAA scorecard JSON + Markdown
   - Flag if below threshold (box IoU <0.65, kappa <0.6)

4. split_dataset.py — train 70% / val 15% / test 15% with temporal separation
