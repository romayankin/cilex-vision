# P1-V01: Edge Agent

You are working in /repo/services/edge-agent/. Create a Python service:

- main.py — entry point, loads config, starts per-camera pipelines
- camera_pipeline.py — GStreamer decode → motion detector → NATS publisher
- motion_detector.py — frame differencing, configurable sensitivity, scene-change detection
- rtsp_client.py — exponential backoff reconnection (1-60s), decoder error detection, per-camera health scoring
- nats_publisher.py — publish FrameRef protobuf to NATS JetStream (site.{site_id}.frames), three timestamps
- local_buffer.py — ring buffer on disk (10GB default), replay with rate limit on WAN recovery
- config.py — Pydantic Settings from YAML

Dependencies: gst-python, nats-py, protobuf, pydantic, prometheus-client
Metrics: camera_uptime_ratio, decode_errors_total, motion_frames_total, static_frames_filtered_total, nats_publish_latency_ms, buffer_fill_bytes
Tests: test_motion_detector.py, test_rtsp_client.py (mock disconnect), test_local_buffer.py
Dockerfile: python:3.11-slim with GStreamer packages
