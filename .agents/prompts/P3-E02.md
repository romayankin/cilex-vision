# P3-E02: Shadow Comparison Dashboard

You are working in /repo/infra/grafana/. Create Grafana dashboard for shadow vs production metrics.

## Context

During Stage 2 shadow deployment (P3-V02), operators need to compare shadow model performance against production in real time.

## Reading Instructions

Before writing any code, read:

## Implementation

1. Create infra/grafana/dashboards/shadow-comparison.json:
   - Detection count: production vs shadow per camera per minute
   - Confidence distribution: side-by-side histograms
   - Class distribution: production vs shadow
   - Disagreement rate: % of frames where models disagree on detections
   - Latency comparison: production vs shadow p50/p95/p99
   - Sample disagreement viewer: links to debug traces for disagreements
2. Create infra/prometheus/rules/shadow-alerts.yml:
   - Shadow detection count divergence > 20% from production
   - Shadow latency > 2x production
