# P2-A01: Cross-Camera Annotation

You are working in /repo/scripts/annotation/. Set up CVAT projects for Re-ID ground truth.

## Context

MTMC evaluation (P2-E02) needs ground truth cross-camera identity pairs. This task creates CVAT annotation projects and guidelines for cross-camera labeling.

## Reading Instructions

Before writing any code, read:
- .agents/handoff/P1-A01.md — existing CVAT setup, annotation guidelines, IAA scripts
- .agents/handoff/P0-D05.md — topology graph, camera adjacency
- docs/annotation-guidelines.md — existing labeling guide
- CONVENTIONS.md

## Implementation

1. Extend docs/annotation-guidelines.md with cross-camera Re-ID section:
   - How to label the same person/vehicle across multiple camera views
   - Identity numbering scheme (global_id prefix per annotator)
   - Quality criteria for Re-ID pairs: clear view, no heavy occlusion, temporal plausibility
2. Create scripts/annotation/setup_reid_projects.py:
   - CVAT API: create project "reid-eval" with cross-camera identity labels
   - Upload synchronized clips from adjacent cameras
   - Create tasks pairing overlapping time windows from camera pairs
3. Create scripts/annotation/export_reid_pairs.py:
   - Export annotated identity pairs as JSON: [{global_id, camera_id, local_track_id, timestamp, crop_uri}]
   - Compute cross-annotator agreement for identity labels
4. Create scripts/annotation/sample_reid_candidates.py:
   - Query DB for track exits/entries on adjacent cameras within transit-time window
   - Present candidate pairs to annotators for identity confirmation
