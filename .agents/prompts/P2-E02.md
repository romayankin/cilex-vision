You are working in /repo/scripts/evaluation/. Measure cross-camera Re-ID accuracy.

## Context

With MTMC service (P2-V02) running on the pilot site, we need to measure association quality against ground truth cross-camera identity annotations (P2-A01).

## Implementation

1. Export ground truth cross-camera identities from CVAT
2. Query global_tracks and global_track_links from DB for the evaluation time window
3. Compute metrics:
   - Rank-1 accuracy: % of queries where correct identity is top match
   - Rank-5 accuracy
   - mAP (mean average precision) for Re-ID
   - False positive rate: incorrect associations / total associations
   - False negative rate: missed associations / total true associations
   - Per-camera-pair accuracy breakdown
4. Generate evaluation report with pass/fail thresholds (Rank-1 > 70% for go-live)

Files to create:
- scripts/evaluation/run_mtmc_eval.py — evaluation harness
- scripts/evaluation/export_reid_gt.py — export ground truth from CVAT
- scripts/evaluation/reid_metrics.py — metric computation functions
- docs/evaluation-results/mtmc-evaluation.md — results + go/no-go recommendation
