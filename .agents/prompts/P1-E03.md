# P1-E03: Edge Filter Pass-Through Rate Measurement

You are working in /repo/scripts/calibration/. Create:

1. edge_filter_calibration.py — for a specified camera:
   - Signal edge agent to disable motion filtering (NATS command site.{site_id}.control.{camera_id}.calibrate)
   - For 10 minutes, collect ALL frames centrally
   - Run detector on every frame, record which have objects of interest
   - Re-run motion detector offline on same frames
   - Compute: miss_rate, false_trigger_rate, pass_through_rate
   - Output: per-camera calibration scorecard (JSON + Markdown)
   - Update scripts/cost-model/params.yaml with measured pass-through rate

2. schedule_calibration.py — rotate through all cameras, one per hour. Rate-limit to avoid overloading central inference.

Metrics: per_camera_pass_through_rate, calibration_freshness_hours.
