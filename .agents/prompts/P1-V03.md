# P1-V03: Central Decode & Frame Sampling Service

You are working in /repo/services/decode-service/. Python service.

- main.py — Kafka consumer group (frames.sampled.refs), spawns decode workers.
- decoder.py — GStreamer pipeline: fetch frame from MinIO → decode (H.264, H.265, MJPEG) → normalize to RGB → resize to inference resolution (configurable, default 1280x720).
- color_space.py — detect input color space (BT.601 vs BT.709 from stream metadata), convert to consistent RGB. Critical: wrong color space silently degrades model accuracy.
- sampler.py — configurable FPS (default 5), skip frames to match target rate. Attach camera metadata.
- publisher.py — publish inference-ready FrameRef to Kafka topic.
- config.py — Pydantic Settings.

Metrics: decode_errors_total{codec}, frames_decoded_total, frames_sampled_total, decode_latency_ms.
Tests: test_decoder.py (H.264/H.265 fixtures), test_color_space.py (BT.601→RGB, BT.709→RGB), test_sampler.py.
Dockerfile with GStreamer packages.
