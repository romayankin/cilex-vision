# P0-O01: Infrastructure Scaffolding

You are working in /repo. Create the development infrastructure:

1. infra/docker-compose.yml — Full local stack:
   - Kafka (3 brokers via bitnami/kafka) + kafka-ui
   - NATS (with JetStream enabled)
   - TimescaleDB (timescale/timescaledb:latest-pg16)
   - MinIO (with auto bucket creation: frame-blobs, event-clips, debug-traces, thumbnails)
