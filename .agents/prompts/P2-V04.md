# P2-V04: Clip Pipeline

You are working in /repo/services/clip-service/. Create a Python service.

## Context

The Clip Pipeline extracts video clips triggered by events from the Event Engine (P2-V03). It uses FFmpeg to extract segments from stored video and uploads them to MinIO.

## Implementation

1. Consume from `events.raw` Kafka topic
2. For each event, determine clip window: event.start_time - 5s pre-roll to event.end_time + 5s post-roll (configurable)
3. Locate source video frames in MinIO by camera_id + time range
4. Extract clip using FFmpeg subprocess: re-encode to H.264 baseline at reduced bitrate (1.5 Mbps)
5. Generate thumbnail: extract middle frame, resize to 320x180
6. Upload clip and thumbnail to MinIO bucket `event-clips` with key: `{site_id}/{camera_id}/{date}/{event_id}.mp4`
7. Update events table: set clip_uri and thumbnail_uri
8. Publish completion to `archive.transcode.completed`
